# VOD-VIP 视频会员系统 — 开发记录

## 项目概述

基于 **Laravel + Vue 3 + Tailwind CSS** 的视频会员系统，支持内容付费、VIP 会员、Telegram Bot 资源采集、后台管理等功能。

---

## 一、项目基础搭建

- Laravel 后端 + Vue 3 前端 + MySQL + Redis 全栈架构
- 数据库设计：用户、视频、分类、订单、VIP 套餐等表
- 用户角色体系：游客、普通用户、VIP 会员、管理员
- Laravel Sanctum API 认证

## 二、核心业务功能

- **视频系统**：分类浏览、视频列表、视频播放（Artplayer）
- **VIP 会员**：VIP 套餐展示、下单购买、VIP 到期管理、VIP 内容拦截
- **收藏功能**：收藏 / 取消收藏视频（仅限 VIP 用户）

## 三、Telegram Bot 资源采集

- Telegram Bot 自动接收媒体文件并下载保存
- `media_resources` 表存储元数据
- 支持 Telegram Bot API Local Server（突破 20MB 文件限制至 2GB）
- 媒体资源可同步为视频条目

## 四、后台管理系统

- 管理员权限中间件 + `is_admin` 字段
- 仪表盘（统计数据）
- 视频、分类、用户、订单、媒体资源的 CRUD 管理

## 五、首页与浏览体验

- 首页重新设计：**最新更新** + **为你推荐** + **分类浏览**
- 独立的 **分类浏览页** (`/browse`)：先展示分类卡片，点击后显示视频列表
- 独立的 **搜索页** (`/search`)：顶部导航搜索按钮 + 首页搜索框均跳转至此

## 六、UI / 导航调整

- 顶部导航：去掉后台入口、VIP 入口重新添加、Logo 点击回首页
- 底部 Footer：去掉快捷链接，简化布局
- 分类浏览页：去掉「返回分类」按钮和搜索栏
- 首页分类区增加「全部」按钮
- 「查看全部」链接指向正确的视频列表页

## 七、注册与登录系统

- **注册页** (`/register`)：用户自填用户名 + 密码 + 确认密码
- **快捷注册**（登录页底部）：一键点击，系统随机生成用户名和密码，弹窗展示并支持一键复制
- **登录页** (`/login`)：用户名 + 密码登录
- 两种注册方式均支持注册后自动登录
- 邮箱字段改为可选（nullable）

## 八、Bug 修复

- 前端 MP4 播放失败（种子数据死链 + HTTPS 混合内容 + 播放器初始化时机）
- Admin 页面刷新跳回首页（`fetchUser` 竞态条件）
- 收藏状态不正确（列表页 prop 响应性 + 播放页 Auth guard 问题）
